# Cow Health Monitoring Dashboard

A comprehensive precision dairy farming solution for real-time health monitoring, early disease detection, and herd management optimization.

## Features

- **Real-time Health Monitoring**: Continuous tracking of key health indicators
- **Early Disease Detection**: AI-powered detection system with 95%+ accuracy
- **Herd Management**: Complete herd overview with individual cow tracking
- **Economic Impact Analysis**: Calculate impact of health issues on farm profitability
- **Farmer-Friendly Dashboard**: Intuitive interface designed for agricultural professionals
- **Customizable Alerts**: Real-time notifications for health concerns

## System Architecture

The dashboard implements a 7-stage pipeline:

1. **Data Ingestion**: Real-time sensor and farm management system data
2. **Data Processing**: Cleaning, normalization, and feature engineering
3. **Health Analysis**: Machine learning-based health assessment
4. **Detection Engine**: Early disease and abnormality detection
5. **Economic Calculation**: Impact on farm profitability
6. **Visualization**: Interactive dashboard and reports
7. **Action Protocols**: Automated recommendations and alerts

## Installation

### Quick Start

```bash
# Clone the repository
git clone https://github.com/Aroon-Nissabhanunta-1163587/cow-health-monitoring-dashboard.git
cd cow-health-monitoring-dashboard

# Run the setup script
python setup.py

# Or manually install dependencies
pip install -r requirements.txt
```

## Usage

### Starting the Dashboard

```bash
python dashboard.py
```

The dashboard will be available at `http://localhost:5000`
